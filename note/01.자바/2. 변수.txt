2. 변수

- 프로그램 작업을 하기위해서 하나의 값을 저장할 수 있는 메모리
- 임의 공간 저장 가능
- 자바에 명령 입력시 ex) int = 데이터 타입, i = 변수명, =, 할당연산자, 10; = 변수할 내용물이다.
- 논리형, 문자형, 정수형, 실수형의 기본 자료형이 있다
- 형변환 방식은 묵시적 형변환, 명시적 형변환이 있으며 묵시적 형변환은 '작은 주머니의 데이터를 큰 주머니로 옮기는 것', 명시    적 형변환은 '큰 주머니의 데이터를 작은 주머니로 옮기는 것'인데 명시적 형변환은 '데이터 손실에 우려가 있으니 코드에 명시     해야만 한다
- Primitive 기본 자료형 = (메모리에 있는 실제 값 = 변수 데이터 값)
- Object 객체 자료형 (참조 데이터 자료형) = (메모리에 있는 실제값 = 변수가 저장된 주소(실제 변수 데이터는 heap 영역)


정리: 변수 입력시 변수명은 자신이 정할 수 있으며 예시로 숫자는 num, 월은 mon, 과목은 n 뒤에 Kor 또는 Mat, Eng를 붙여 넣을 수 있다.
